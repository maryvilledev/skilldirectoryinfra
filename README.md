# Devstack

### To Run Locally
`./devstack`

### Pull newest images
`./devstack pull`

### Only start Cassandra
`./devstack data`

### Stop and remove skilldirectory and skilldirectoryui containers
`./devstack stop`

### Stop and remove all containers
`./devstack kill`

### Run UI with dev API
`./devstack --dev`

### Drop Cassandra data locally
`./devstack --dropdata`
